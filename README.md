# Exploratory distribution analysis of Stryphnodendron confertum



This repository contains an exploratory spatial analysis of occurrence records of *Stryphnodendron confertum* in central Brazil.



The project aims to:



* compile cleaned occurrence records
* visualize geographic distribution
* calculate a convex hull approximation of extent of occurrence
* identify potential spatial gaps in records

The observed distribution gap may be explained by:

1. collection bias
2. environmental constraints
3. potential disjunct populations



This repository represents an exploratory analysis and not a definitive biogeographic conclusion.



Issues that need to be considered are:



* Only species links records were used so forth
* A broader evaluation of the nature of the gap in the records is required, through the proper phytogeographical techniques
* A proper ecological model to adress the problem needs to be developped and so on...
